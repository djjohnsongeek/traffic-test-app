Goals

The current consensous, or at least it seems to me, is that when you are creating a new website the best way to generate traffic is to pay money for advertisements, or to pay content farms for views. This website is an experiament to test an alternate theory.

Is is possible to generate sufficient web traffic for an unknown website simply by regularly posting (not spamming) on social sites such as Reddit or Discord? I don't actually know. Others I have talked to are pretty doubtful. But as one indiviudal wisely asked:

Why don't you test that theory out?

So here we are. The plan is to post a link to this website once per day on a popular social platform for about two weeks (or more if it gets interesting). The website will record views, and the data will be visible for all on the reports page.
